# Mystery Phone

A browser-based found-phone mystery game built with Vite, React, and PWA support.

The landing page explains the concept at `/`. The playable case is available at `/play`.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Case One

`The Last Platform` asks the player to investigate Lena Ortiz's recovered phone, cross-reference apps, solve evidence puzzles, reconstruct her final timeline, and make a final deduction.
